## Running locally

1. Clone this repo:

```sh
$ git clone https://github.com/ykhost/geo_capital_portfolio_de_empresas.git
```

2. Then go to the project's folder:

```sh
cd geo_capital_portfolio_de_empresas
```

3. Install all dependencies:

```sh
yarn install
```

4. Run locally:

```sh
yarn start
```
